# Code Documentation Assistant Backend

Backend service for the Code Documentation Assistant, built with FastAPI, Temporal, and LangGraph.

## Features

- **FastAPI**: Modern, fast (high-performance) web framework (based on Starlette and Pydantic) for building APIs with Python 3.6+ based on standard Python type hints.
- **Temporal**: Workflow orchestration platform for running background tasks and workflows.
- **LangGraph**: Framework for building stateful, multi-actor applications with LLMs.
